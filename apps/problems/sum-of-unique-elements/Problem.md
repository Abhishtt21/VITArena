### Sum of Unique Elements

Given an array of integers, return the sum of all unique elements in the array. An element is considered unique if it appears exactly once in the array.


#### Test case 1

Input

```
[1, 2, 3, 2]
```

Output

```
4
```


#### Test case 2

Input

```
[4, 4, 5, 6]
```

Output

```
11
```


#### Test case 3

Input

```
[7, 7, 7, 7]
```

Output

```
0
```
